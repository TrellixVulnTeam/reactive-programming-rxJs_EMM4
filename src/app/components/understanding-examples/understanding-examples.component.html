Operator Take

Operator Take gets count of taking emitted values as a parameter and
after taking all of them gets completing. In this case you can find
bug if you will click on button several times.


You can import operator take from rxjs/operators library

Operators Take and TakeUntil

Operator TakeUntil gets observable as a parameter and takes values
until hasn't called next function of took parameter.
In this example we are calling next function of an observable set
as a parameter on every start button click. In this case stream
finishes and starts over.


You can import operator takeUntil from rxjs/operators library
